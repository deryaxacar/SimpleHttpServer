About This Project

This HTTP server was built using C programming language.

It serves static files like HTML, CSS, and JavaScript from a directory.

Project Features

  • Handles multiple client requests concurrently.
  • Supports basic HTTP methods: GET and POST.
  • Serves static files including images and documents.
  • Provides error handling for missing files.

Installation and Usage

To run the server, follow these steps:

  1. Clone the repository from GitHub.
  2. Compile the source code using GCC: gcc -o http_server simple_http_server.c -lpthread
  3. Run the server: ./http_server
  4. Open your web browser and go to http://localhost:8080.
